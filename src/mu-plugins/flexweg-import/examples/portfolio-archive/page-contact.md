# Contact

The fastest path is email — `hello@example.com`. We answer within one working day for projects under €25k, two days for larger work.

## The studio

> 27 Rue des Tournelles  
> 75003 Paris, France

We don't keep walk-in hours, but if you're in the neighbourhood with a project in mind, send us a note first and we'll usually find an hour.

## What we need to start a conversation

- A short brief — three or four sentences of what you have in mind
- Timeline — when you'd like the work delivered
- Budget range — broad bracket is fine, we'll tell you whether we can fit before any detailed pricing

We turn down most inquiries, not from arrogance but from capacity. If we say yes, we say yes for the full project.
