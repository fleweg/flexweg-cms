## Description

The minimum-viable SEO layer for any Flexweg CMS site. On every published page, Core SEO injects:

- `<meta name="twitter:card" content="summary_large_image" />` plus the matching `twitter:image` / `twitter:title` / `twitter:description` derived from each post's frontmatter
- `<meta name="generator" content="Flexweg CMS" />` so monitoring tools can detect your platform

Built-in. No settings page. Toggle it on and forget about it.
